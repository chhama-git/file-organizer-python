# File Organizer Tool

This project organizes files into folders automatically using Python.

## Features
- Sorts images and PDFs
- Easy to use

## How to run
python main.py

## Python Concepts Used
- File handling using os module
- Directory operations (os.listdir, os.path.exists, os.mkdir)
- Path management using os.path.join
- Conditional statements (if-elif)
- Loops (for loop)
- String methods (endswith)
- File moving using shutil module

## Features
- Automatically organizes files into folders
- Separates image files (.jpg, .png)
- Separates PDF files (.pdf)
- Creates folders if they do not exist
- Skips directories (only processes files)
- Simple and beginner-friendly script
